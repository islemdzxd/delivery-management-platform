# Transport Delivery Frontend

A React application with Tailwind CSS for managing transport and delivery operations.

## Features

- Client Management
- Expedition Tracking
- Driver Management
- Vehicle Management
- Destination Management
- Service Type Management

## Setup

1. Install dependencies:
```bash
npm install
```

2. Make sure the Django backend is running on `http://localhost:8000`

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
```

## Technologies

- React 18
- Vite
- Tailwind CSS
- Axios
- React Router DOM
- React Icons
